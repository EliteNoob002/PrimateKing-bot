# PrimateKing-bot

Discord бот для сервера "Отлов приматов" с расширенным функционалом.

## Описание

PrimateKing-bot — многофункциональный Discord бот, предоставляющий широкий спектр возможностей включая интеграцию с Yandex GPT, генерацию изображений, управление базой данных пользователей, SSH операции и многое другое.

## Основные функции

- **Yandex GPT интеграция** — возможность задавать вопросы и получать ответы от Yandex GPT
- **Генерация изображений** — создание изображений на основе текстовых промптов через Yandex GPT Art
- **Управление пользователями** — система подсчёта и статистики для пользователей сервера
- **Slash команды** — современные интерактивные команды Discord
- **Prefix команды** — классические команды с префиксом
- **Логирование** — отслеживание удалённых сообщений и действий пользователей
- **SSH интеграция** — удалённое управление сервером через SSH
- **Telegram уведомления** — отправка уведомлений о важных событиях в Telegram
- **API синхронизация** — интеграция с внешним API для управления командами
- **Ротация статусов** — автоматическая смена статуса бота
- **Прокси поддержка** — работа через прокси для всех Discord запросов

## Используемые технологии

- **Python 3** — основной язык программирования
- **discord.py** — библиотека для работы с Discord API
- **Yandex GPT API** — интеграция с Yandex GPT и GPT Art
- **MySQL** — база данных для хранения информации о пользователях
- **aiohttp** — асинхронные HTTP запросы
- **paramiko** — SSH клиент для удалённых операций
- **PyYAML** — работа с конфигурационными файлами

## Структура проекта

Проект организован в модульную структуру для удобства разработки и поддержки:

- `commands/` — обработчики команд (slash и prefix)
- `events/` — обработчики событий Discord
- `services/` — интеграции с внешними сервисами
- `utils/` — вспомогательные утилиты и функции
- `tasks/` — фоновые задачи
- `ui/` — компоненты пользовательского интерфейса Discord
