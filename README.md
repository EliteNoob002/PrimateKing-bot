# 🤖 PrimateKing-bot

**Discord бот для сервера "Отлов приматов"**

Многофункциональный Discord бот с интеграцией Yandex GPT, генерацией изображений, управлением базой данных пользователей и другими возможностями.

---

## ✨ Основные возможности

### 🤖 Искусственный интеллект
- **💬 Yandex GPT интеграция** — задавайте вопросы и получайте ответы от Yandex GPT
- **🎨 Генерация изображений** — создание уникальных изображений на основе текстовых промптов через Yandex GPT Art
- **🔄 Интерактивные UI** — кнопки для скачивания, копирования промптов и повторной генерации изображений

### 📊 Управление пользователями
- **👥 Система статистики** — отслеживание активности и подсчёт действий пользователей
- **💾 MySQL база данных** — надёжное хранение данных через SQLAlchemy ORM
- **📈 Детальная статистика** — информация о пользователях и их активности

### 🎮 Команды
- **⚡ Slash команды** — современные интерактивные команды Discord с автодополнением
- **⌨️ Prefix команды** — классические команды с настраиваемым префиксом
- **🎯 Управление через web-панель** — включение и отключение команд без рестарта бота

### 🔧 Дополнительные функции
- **📝 Логирование** — отслеживание удалённых сообщений и действий пользователей
- **🔌 SSH интеграция** — удалённое управление сервером через SSH
- **📱 Telegram уведомления** — отправка важных событий в Telegram канал
- **🔄 Ротация статусов** — автоматическая смена статуса бота
- **🌐 Прокси поддержка** — работа через прокси для всех Discord запросов

---

## 🛠 Технологии

- **Python 3.8+** — основной язык программирования
- **discord.py** — библиотека для работы с Discord API
- **Yandex GPT API** — интеграция с Yandex GPT и GPT Art для генерации текста и изображений
- **SQLAlchemy** — ORM для работы с MySQL базой данных
- **aiohttp** — асинхронные HTTP запросы
- **paramiko** — SSH клиент для удалённых операций

---

## 📝 Доступные команды

### Slash команды
- `/help` — список доступных команд
- `/gpt <запрос>` — задать вопрос Yandex GPT
- `/gpt_art <промпт>` — сгенерировать изображение по текстовому описанию
- `/avatar [участник]` — получить аватар участника сервера
- `/count [участник]` — узнать статистику участника
- `/send_blin [участник] [текст]` — отправить "блин с говном"
- `/sas` — развлекательная команда
- `/poslat [участник]` — отправить участника
- `/restartbot` — перезапустить бота (только для администраторов)
- `/update` — обновить файлы бота (только для администраторов)

### Prefix команды
- `$teste` — тестовая команда с SSH
- `$testl` — тестовая команда
- `$testo` — тестовая команда
- `$say` — отправить сообщение (только для владельца)

## 🏗 Архитектура

Бот построен на модульной архитектуре:

- **commands, events, services, utils, models** — разделение по зонам ответственности
- **ConfigCache** — быстрый доступ к настройкам из общей с панелью MySQL
- **Web-панель** — редактирование prefix, команд, статусов и других runtime-настроек
